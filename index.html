Projects

Cutesy

Live Site | Github

Cutesy is a fullstack clone of the popular online global marketplace, Etsy. The application was built using React, Redux, Javascript, Ruby, Rails, and PostgreSQL. With Cutesy, users are able to browse through many cute products. This application utilizes RESTful routes to implement CRUD functionality and allows for dynamic interaction with the application, such as adding & removing of products to one's cart.

LearnTwoMath

Live Site | Github

LearnTwoMath is an online platform for users to either share resources about mathematical concepts or learn from the resources posted using Node.js, Express, MongoDB, React, Redux, Quill.js, Javascript, and Webpack.

Madlibs Against Humanity

Live Site | Github

Madlibs Against Humanity is a website where users can create, play, and review madlibs, and search for madlibs based on certain categories created using React, Redux, Javascript, NodeJS, Express, MongoDB, Mongoose, Express.js, and Axios.
